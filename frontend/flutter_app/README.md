# GLIDE Flutter App

Flutter app integrating Firebase and GLIDE SearchService.

## Setup
1. Copy `.env.example` to `.env` and set `BACKEND_BASE_URL`.
2. Replace `lib/firebase_options.dart` with your Firebase config (FlutterFire CLI).
3. Run:
```bash
flutter pub get
flutter run
```

## Screens
- Login/Register
- Home (Search)
- Available Rides List
- Ride Details
- My Rides
- Chat
